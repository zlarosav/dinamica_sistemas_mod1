# Dinámica de Sistemas - Módulo 1

Repaso estático para las semanas 1, 2 y 3 del curso de Dinámica de Sistemas.

El sitio publicado muestra todas las preguntas por semana, con alternativas, respuestas ocultas por defecto, argumentos ampliados y referencia directa a la diapositiva del PDF teórico correspondiente.

## Desarrollo

Para regenerar el sitio local:

```powershell
node scripts/build-study-pages.js
```

GitHub Pages publica el contenido estático de `build/` desde la rama `gh-pages`.
